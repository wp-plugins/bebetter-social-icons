BeBetter Social Icons is a simple, light weight plugin which allows you to easily add social icons to your site.

== Description ==

It is a fork of WP Social Icons from http://magnigenie.com/wp-social-icons-easily-add-social-icons-site/, adapted to the specific needs of our business.

= Features: =

* Choose from 202 font icons.
* Easily change color of icons to match your website look and feel.
* Fully responsive front end.
* Easy to use admin interface.
* Easily add any social network based on the icons.
* Uses shortcode to display your social icons on your site.
* Adds ability to put shortcodes on widgets too.

== Installation ==

1. Go to your admin area and select Plugins -> Add new from the menu.
2. Search for "BB Social Icons".
3. Click install.
4. Click activate.
5. Go to Settings > BB Social Icons and start playing with the configurations.
